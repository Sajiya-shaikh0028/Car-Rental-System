#Car Rental System (Java)#

A console-based *Car Rental System* built using *Java* and *Object-Oriented Programming (OOP)* concepts.  
The application allows users to rent and return cars through a simple menu-driven interface.

#Features:
- View available cars
- Rent a car for a specific number of days
- Return a rented car
- Automatic price calculation
- Menu-driven console interface
- Object-oriented design (Car, Customer, Rental classes)
  

#Technologies Used:
- Java
- VS Code
- Java Collections (ArrayList)
- Scanner class for user input
